# Personal Website

My personal portfolio website — a modern, responsive single-page site showcasing my bio, experience, projects, and ways to get in touch.

Built with vanilla HTML, CSS, and JavaScript. Features a dark theme, scroll animations, and a clean mobile-responsive layout.

## Sections

- **Hero** — Introduction with name, tagline, and social links
- **About** — Bio and technology skills
- **Experience** — Work history timeline
- **Projects** — Featured project cards
- **Contact** — Get in touch call-to-action

## Live Site

[Visit the site →](https://ssimkani.github.io/personal-website)

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, custom properties, animations)
- JavaScript (Intersection Observer, smooth scroll)
- Font Awesome & Google Fonts
